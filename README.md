# Dashboard de Métricas en Tiempo Real (Angular + TypeScript)

Aplicación de dashboard construida con **Angular** que obtiene datos desde una **API simulada** y los muestra en:
- Un **gráfico de barras** (tareas completadas por proyecto)
- Un **gráfico de líneas** (bugs reportados por proyecto)

## Tecnologías
- Angular (Standalone)
- TypeScript
- chart.js
- ng2-charts

## Requisitos
- Node.js (recomendado: LTS)
- Angular CLI

## Instalación
1. Clona el repositorio:
   git clone <URL_DEL_REPOSITORIO>
   cd dashboard-proyecto
